# Fretish Requirements for VALU3S Use Cases 5
#### Matt Luckcuck December 2025

This directory contains the FRETISH requirements for [Use Case 5](https://repo.valu3s.eu/use-cases/aircraft-engine-controller) from the VALU3S project. 

* `UC5 Original Requirements.pdf` - The Natural-Language Requirements and Test Cases Use Case 5
* `EngineControllerv2.1` - The requirements set for Use Case 5, before refactoring
* `EngineControllerv2.1.1.json` - The requirements set for Use Case 5, before refactoring but after merging the child requirements in preparation for refactoring 
* `EngineControllerv2.2.json` - The requirements set for Use Case 5, after refactoring
